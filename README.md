<h2>Dag 5 (12-03-2026)</h2>
<h3>Wat heb ik gedaan vandaag?</h3>
<p>Ik ben begonnen met het maken van animaties voor mijn radio. Ik heb uit keyframes gebruikt met dat de vorm transform. op en neer. Zo lijkt het alsof er muziek uit de radio komt. Er was alleen een probleem met hoe de animatie bewoog. Het bewoog via het midden dus hij stretchen over de tafel heen. Dit was een makkelijke fix. Als je de transform origin op bottom zet blijft het ope een vaste plek. Na dat ik dat heb gedaan ben ik begonnen met de twee themes. Ik dacht aan disco thema en een normale thema. Dit pasten goed bij de vibe. Ik heb weer een keyframe gemaakt met nu verschillende kleuren en dit op de main gezet zodat de hele scherm een disco licht effect heeft. Daarna heb ik met een clippath de disco effect afgesneden naar een driehoek dat lijkt op een spotlight. Als laatst heb ik de disco light effect gekoppeld met een light mode dark mode toggle.  </p>
<h3>Hoelang heb ik er aan gewerkt?</h3>
<p>Ik was de eerste uur bezig met het volgen van de checkin les en vervolgens de hele les bezig geweest met het maken van de animaties. De laatse kwartier was ik bezig met de checkout.</p>
<h3>Wat heb ik geleerd?</h3>
<p>Ik heb geleerd hoe ik keyframes kan gebruiken om animaties te maken. Ik heb ook geleerd hoe je een animaties om 1 plek kan laten bewegen inplaats van dat alles beweegt. Ik heb ook geleerd dat met de hulp van :before en :after dat je extra elementen kunt toevoegen aan je HTML. Ik had zelf niet gedacht dat dat mogelijk zou zijn. </p>
<h3>Wat ga ik morgen doen?</h3>
<p>Morgen ga ik de animaties interactief maken en kijken hoe ik ze kan gebruiken in mijn project, en hoe ik de muziek notes ga maken want ik ben het continue aan het verlengen.</p>
