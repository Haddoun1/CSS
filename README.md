<h2>Dag 1 (04-03-2026)</h2>
<h3>Wat heb ik gedaan vandaag?</h3>
<p>Omdat er niet was vorige week vrijdag moest ik de opdracht bijlezen. Ik mocht kiezen uit 4 verschillende opdrachten en ik heb uiteindelijk gekozen voor de control panel opdracht. Ik ben daarna inspiratie gaan doen op de website Flickr en ik kwam een vintage radio tegen die mij erg interesant leek. Ik had een heel idee over hoe ik het muziek zou laten afspelen en dat er muziek notes uit zouden komen. Alleen is het muziek laten afspelen best moeilijk omdat we geen javascript mogen gebruiken. Dus ik ga nu focussen op dat de radio veel beweegt alsof het muziek afspeelt. Ik wil ook dat er nogsteeds muziek notes uit komen. Ik ben begonnen met de base en heb verschillende divs samen gevoegd zodat het een beetje leek op een radio base. Voor de rest heb ik ook de workshops gevolgd. </p>

<h3>Hoelang heb ik er aan gewerkt?</h3>

<p>Ik heb niet echt iets nieuws geleerd. Vandaag was meer het begin maken en kijken wat ik wou maken als eind opdracht.</p>

<h3>Wat ga ik morgen doen?</h3>
